# VaultLog

A task and time tracking system for managing client projects and team productivity.

## Features

- Task management with priority and status tracking
- Time logging (billable and non-billable hours)
- Client organization and management
- Role-based access control (Admin, Manager, Staff)
- Activity audit trail
- File attachments for tasks

## Tech Stack

Next.js 16, React 19, TypeScript, Tailwind CSS, PostgreSQL (Neon), Drizzle ORM, NextAuth.js

## Quick Start

```bash
npm install
npm run db:migrate
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run db:migrate` - Run migrations
- `npm run db:studio` - Drizzle Studio
- `npm run seed` - Seed sample data
